# 🧟 ZOMBIE HORDE
### Многопользовательский 3D-шутер в браузере

> Защищайте базу от волн зомби вместе с друзьями!
> 10 волн нарастающей сложности, 3 типа врагов, грозный босс Ширибазаров.

---

## 🎮 О игре

**Zombie Horde** — многопользовательский 3D-шутер от первого лица с элементами защиты базы.
Игра запускается прямо в браузере без установки дополнительного ПО.

### Технологии
- 🖥️ **Клиент**: React + TypeScript + Three.js (CDN) + Vite
- 🖧 **Сервер**: Node.js + Express + Socket.IO
- 🌐 **Публичный доступ**: ngrok-туннель
- 🎨 **Графика**: Three.js r128, 3D-примитивы, частицы, тени

## 🌐 Мультиплеер — подключение

### Одиночная игра
1. Нажмите **«Одиночная игра»**
2. Игра запустится сразу (все механики доступны)

### Мультиплеер
1. Запустите сервер (`node server.js`) — см. [server/SERVER.md](server/SERVER.md)
2. Скопируйте WebSocket URL из консоли: `wss://xxxx.ngrok-free.dev/ws/`
3. Нажмите **«Подключиться к серверу»**
4. Вставьте URL в поле ввода
5. Нажмите **«Подключиться»**
6. Поделитесь URL с друзьями — они делают то же самое

## 🚀 Развёртывание

## 🎨 Графика и эффекты

- **Three.js** — 3D-рендеринг
- **Тени** — DirectionalLight с PCFSoftShadowMap
- **Туман** — FogExp2 для атмосферы
- **Частицы**: брызги крови, взрывы, искры, кислота
- **Мини-карта** — 2D Canvas-оверлей
- **HUD**: здоровье, патроны, деньги, волна, комбо, усиления
- **Анимации**: покачивание оружия, анимации зомби, эффекты усилений

## 🐛 Известные особенности

- В одиночном режиме все механики работают локально
- При большом количестве игроков (20+) возможно снижение FPS — это нормально
- ngrok бесплатный туннель имеет ограничения по полосе пропускания
- Для продакшена рекомендуется VPS с внешним IP
