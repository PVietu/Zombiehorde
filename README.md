# 🧟 ZOMBIE HORDE v2.0

**Многопользовательский 3D-шутер в браузере с защитой базы**

> Защищайте базу от волн зомби вместе с друзьями! 10 волн нарастающей сложности, 5 типов врагов, грозный босс Ширибазаров и десятки игровых механик.

---

## 🎮 Об игре

**Zombie Horde** — это браузерный 3D-шутер от первого лица в жанре "выживание против волн зомби". Игра использует Three.js для 3D-рендеринга и Socket.IO для мультиплеера в реальном времени.

### Технологии

- 🖥️ **Клиент**: React + TypeScript + Three.js (CDN) + Vite
- 🖧 **Сервер**: Node.js + Express + Socket.IO
- 🌐 **Публичный доступ**: ngrok-туннель
- 🎨 **Графика**: Three.js r128, 3D-примитивы, частицы, тени

## 🧟 Типы врагов

| Тип | HP | Урон | Особенности |
|-----|----|------|-------------|
| **Обычный** | 80 | 10 | Медленный, рукопашный |
| **Взрывающийся** | 60 | 80 (АОЕ) | Взрывается при приближении/смерти |
| **Кислотный** | 70 | 15 | Дальнобойный, оставляет лужу |
| **Минибосс** | 300+ | 20+ | 1-2 случайных модификатора |
| **Ширибазаров** | 3000 | 40 | 3 атаки, убийство = победа |

### 🦠 Модификаторы минибосса

- **Огромный** — ×2 HP, увеличен размер
- **Быстрый** — скорость +40%
- **Ядовитый** — атаки накладывают яд
- **Бронированный** — урон −30%
- **Регенерирующий** — восстанавливает HP без урона
- **Взрывной** — взрывается при смерти
- **Телепортирующийся** — телепортируется к игроку каждые 8 сек
- **Призыватель** — призывает 2 зомби каждые 12 сек

## 🚀 Запуск игры

### Одиночная игра (Solo)

```bash
# 1. Установить зависимости
npm install

# 2. Запустить dev-сервер
npm run dev

# 3. Открыть в браузере http://localhost:5173
# 4. Нажать "Одиночная игра"
```

### Мультиплеер

#### Шаг 1: Запустить игровой сервер

```bash
cd server/
npm install
node server.js
```

Сервер выведет публичный URL через ngrok, например:
```
🌐 WebSocket URL: wss://xxxx.ngrok-free.dev
```

#### Шаг 2: Подключиться

1. Открыть игру в браузере
2. Нажать **"Подключиться к серверу"**
3. Вставить WebSocket URL из консоли
4. Нажать **"Подключиться"**
5. Поделиться URL с друзьями

## 🎨 Графика и эффекты

- **Three.js** — 3D-рендеринг
- **Тени** — DirectionalLight с PCFSoftShadowMap
- **Туман** — FogExp2 для атмосферы
- **Частицы**: брызги крови, взрывы, искры, кислота
- **Мини-карта** — 2D Canvas-оверлей
- **HUD**: здоровье, патроны, деньги, волна, комбо, усиления
- **Анимации**: покачивание оружия, анимации зомби, эффекты усилений
- **Дульная вспышка** и трассеры пуль
- **Фейерверки** при победе
